FAQ

What is a Genesis Agent?

+

Genesis Agents are autonomous, cross-chain programs capable of executing complex tasks across blockchains and dApps — without user intervention. They're persistent, composable, and intelligent.

How are agents different from smart contracts or bots?

+

Smart contracts are static and reactive. Bots are off-chain and limited. Agents combine the best of both — they live on-chain, reason off-chain, and act autonomously.

What chains do Genesis Agents support?

+

Genesis supports Ethereum, Solana, Layer 2s (Arbitrum, Optimism), BNB Chain, Cosmos-based chains, and more via modular adapters.

Can I build my own agent?

+

Yes. With the Genesis SDK, anyone can create and deploy a custom agent using pre-built modules or custom logic.

Are Genesis Agents secure?

+

Yes. Agent execution is sandboxed, cryptographically verifiable, and non-custodial. All interactions are recorded on-chain and agents operate within predefined permissions.
